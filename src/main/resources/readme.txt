
Hierbei handelt es sich um ein Web Entwicklungsprojekt
im Rahmen vom WIFI Kurs "Software Developer Java"

Aufgabenstellung:

Implementieren Sie eine Webapplikation für die Brechnung eines Rechtecks.
Dabei müssen folgende Grundfunktionen enthalten sein:
- Datenobjekt
- Eingabe der Seiten a und b
- Berechnungsaktion
- Anzeige der Ergebnisse
        - Umfang
        - Fläche
        - Eingegebene Seiten (a und b)
- Der Webauftritt wird mit Bootstrap optisch ausgerichtet.

Bonuspunkte:
- Ein Bild (Rechteck mit/ohne Formel) wird angezeigt
- Eine extra Infoseite ist vorhanden
